# Campus-Helpdesk-Chatbot-A-Rule-Based-AI-Chatbot-with-a-Tkinter-GUI
This desktop chatbot is built in Python using Tkinter. It uses a rule-based AI system to answer common campus questions about admissions, courses, fees, class timings, the library, and contacts. The GUI provides an interactive experience while explaining responses, making the AI's decision process clear and easy to understand.
